; Hexo 静态博客生成器，专为github博客部署

========== 安装前准备 ==========

1.安装Node（必须）
2.安装Git（必须）

=========== 正式安装Hexo =============

1.新建项目文件：hexo

2.在此项目文件执行如下命令安装Hexo：

npm install -g hexo

3.初始化然后，执行init命令初始化hexo,命令：

hexo init

4.hexo generate （hexo g也可以）

就会把静态文件生成到 public 文件夹

5.hexo server （hexo s也可以）

浏览器输入http://localhost:4000

============ 安装完成 ===========
